# 🍔 Food Delivery Admin Panel

## 📌 Overview

This is the **Admin Panel** for the Food Delivery application.
It allows admins to manage food items and track customer orders.

---

## 🚀 Features

* ➕ Add new food items
* 📋 View all food items
* ❌ Delete food items
* 📦 View customer orders
* 🔄 Update order status

---

## 🛠️ Tech Stack

* React (Vite)
* React Router DOM
* Axios
* React Toastify
* CSS

---

## 📂 Project Structure

```
src/
  ├── components/   # Navbar, Sidebar
  ├── pages/        # Add, List, Orders
  ├── assets/       # Images & icons
  ├── App.jsx       # Main layout & routing
  └── main.jsx      # Entry point
```

---

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/your-username/food-delivery-admin.git
```

### 2. Install dependencies

```
npm install
```

### 3. Run the project

```
npm run dev
```
